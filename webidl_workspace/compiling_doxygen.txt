Compiling Doxygen Source
----------------------------


## DEPENDENCIES

	GNU flex, bison, libiconv, make, strip
	python > 2.6
	cmake > 3.1.3
	Qt > 4.3
	latex
	graph visualization toolkit > 1.8.10 with freetype support


## COMPILATION

	mkdir build
	cd build

	cmake -G "Unix Makefiles" ..
	cmake -Dbuild_wizard=YES ..
	cmake -L ..

	make

	// for manual:
	cmake -Dbuild_doc=YES ..
	make docs


## INSTALLATION

	./configure
	make install
